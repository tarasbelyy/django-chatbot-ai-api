# API сервис для создания, изменения и использования чат-ботов на основе GPT


## Описание
REST API для создания, редактирования и использования чат-ботов с применением технологий искусственного интеллекта OpenAI ChatGPT.
Доступные эндпоинты:
- CRUD операции для ботов:
```
/api/v1/bots/
```
- CRUD операции для сценариев:
```
/api/v1/scenarios/
```
- CRUD операции для шагов сценария:
```
/api/v1/scenarios/<int:scenario_id>/steps/
```
- Запуск бота (GET-запрос) и общение с ботом (POST-запрос):
```
/api/v1/bots/<int:bot_id>/run/
```
- Создание и редактирование данных пользователей:
```
/api/v1/users/
```
- Получение токена для аутентификации:
```
/api-token-auth/
```

## Функциональность
- осуществляется создание, просмотр, изменение и удаление ботов, сценариев, отдельных шагов сценария
- поддерживается запуск бота на основе OpenAI ChatGPT и общение с ботом через API
- осуществляется добавление новых пользователей и получение токена для запросов к API
- один пользователь может запустить несколько разных ботов одновременно
- выполняется проверка корректности сценариев: доступность exit из любой точки сценария, достижимой из start
- в случае ошибок передаются соответствующие сообщения и код ошибки: 400, 401, 403, 404, 500
- настроен CI/CD пайплайн с использованием GitHub Actions для автоматического запуска проекта при обновлениях

## Требования
Основные внешние библиотеки и фреймворки для Python, используемые в проекте: 
- django версии 6.0.6
- djangorestframework версии 3.17.1
- openai версии 2.44.0
- psycopg2-binary версии 2.9.12
- gunicorn версии 26.0.0

Приложение развертывается в Docker.
Для хранения данных используется PostgreSQL.
В качестве веб-сервера используется Nginx.
